## Purpose

This is **Project 6** of Udemy's ["50 Projects in 50 Days for HTML, CSS and JavaScript"](https://www.udemy.com/course/50-projects-50-days/learn/lecture/23595208#overview) taught by Brad Traversy.

This small project creates a webpage where, when scrolled through, continues to load cards as the user skims down the page.
